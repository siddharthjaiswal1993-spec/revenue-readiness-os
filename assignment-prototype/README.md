# Mindtickle Context Intelligence — Assignment Prototype

Standalone high-fidelity prototype for the Group Product Manager Platform interview assignment.

## Demonstrated workflows

- Multi-context content recommendations for an opportunity
- Grounded explanations, confidence, and recommendation evidence
- Context-source exploration and diagnostics
- Enterprise policy and governance controls
- A shared API and MCP delivery contract

## Run locally

```bash
npm install
npm run dev
```

This prototype uses synthetic data and illustrates a proposed product direction. It is not affiliated with or representative of Mindtickle's actual roadmap.
