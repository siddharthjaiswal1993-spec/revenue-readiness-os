# Revenue Readiness OS — Interactive Prototype

React + Vite + TypeScript + Tailwind + Recharts prototype with 10 pages.

## Setup

```bash
npm install
npm run dev
```

## Pages

| Route | Page |
|-------|------|
| `/` | Command Center — org RRI, coaching queue preview, win-rate correlation |
| `/rep-readiness` | Rep Readiness — individual skill scores, radar chart, tier filtering |
| `/roleplay` | AI Roleplay — scenario library, interactive session, debrief |
| `/coaching` | Coaching Queue — prioritized actions, evidence, complete/dismiss |
| `/programs` | Enablement Programs — completion rates, skill lift, certifications |
| `/content` | Content Hub — searchable library with freshness scoring |
| `/skill-gaps` | Skill Gaps — org-level gap analysis, trend indicators |
| `/deals` | Deal Readiness — deal risk signals from rep skill gaps |
| `/analytics` | Analytics — ramp time cohorts, win-rate correlation, program effectiveness |
| `/ai-assistant` | AI Assistant — natural language queries about team readiness |
